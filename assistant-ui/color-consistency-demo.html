红色颜色一致性验证

浅色主题

1. 点踩按钮 (激活状态)

2. 登出按钮 (hover状态)

3. 对话框图标

bg-red-100 + text-red-600

4. 确认登出按钮

深色主题

1. 点踩按钮 (激活状态)

2. 登出按钮 (hover状态)

3. 对话框图标

bg-red-900/30 + text-red-400

4. 确认登出按钮 (修复后)

颜色系统说明

文本颜色:

  • • 浅色主题: text-red-600
  • • 深色主题: text-red-400

背景颜色:

  • • 按钮: bg-red-600 (两个主题相同)
  • • 浅色背景: bg-red-100
  • • 深色背景: bg-red-900/30

✅ 所有红色使用都遵循Tailwind CSS的标准颜色系统，保持了完美的一致性！

🔧 修复：确认登出按钮现在在深色主题下也使用统一的 border-red-600，避免了 oklch 颜色转换问题。
